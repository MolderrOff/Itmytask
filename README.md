# IT-MyTask: Система управления заявками в IT-аутсорсинге

Проект представляет собой демонстрацию навыков **Back-end разработки** на стеке .NET. Фокус сделан на архитектуре серверной части, взаимодействии с реляционными СУБД и развертывании в Linux-среде.

**Демонстрация (Live Demo):** [http://37.143.10.243](http://37.143.10.243)  
*(Развернуто на Ubuntu 22.04 LTS)*

## Ключевой функционал и технологии

*   **Бизнес-логика**: Моделирование процессов взаимодействия сотрудников и клиентов внутри IT-компании.
*   **Стек**: C#, ASP.NET Core, Entity Framework Core.
*   **База данных**: MySQL. Реализована логика **автоматической инициализации**: при первом запуске приложение проверяет наличие БД и самостоятельно создает её структуру (Code First).
*   **Система наполнения (Seeding)**: Реализован эндпоинт `/Work/GetWorks` для быстрой инициализации базы тестовыми данными, что упрощает демонстрацию функционала на пустых окружениях.
*   **Интерфейс**: Реализован базовый функционал CRUD (создание, чтение, обновление, удаление объектов) через Razor Pages/MVC.

## Технические особенности реализации

*   **Deployment**: Опыт работы с Git и развертывания .NET приложений на Linux-серверах.
*   **Управление конфигурацией**: Гибкая настройка строк подключения для работы с различными экземплярами MySQL.
*   **Автоматизация**: Использование механизмов EF Core для миграций и управления схемой данных «на лету».

```bash
# Установка окружения и сборка
sudo apt install -y dotnet-sdk-8.0
dotnet restore
dotnet build

# Запуск приложения на внешнем порту
dotnet run --urls=http://*:80
